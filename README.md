# Code für Software Engineering 2025/2026 Weihnachtsübungsblatt

## Setup:
### Default
compile and run with

`./gradlew run`

clean up

`./gradlew clean`

*Note: If you are using IntelliJ Idea or a similar IDE, these configurations should be loaded automatically and you can just execute them from the side panel*

### For high DPI displays

`./gradlew run -PhighDpi=2`

clean up

`./gradlew clean`
